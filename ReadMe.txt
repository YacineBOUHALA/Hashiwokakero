*************************** Manuel Technique ********************************

1. pour lancer l’application double cliquez sur le fichier .jar fournit, si cela ne marche pas c’est que 	votre pc n’autorise pas l’exécution d’un fichier portant cette extension, dans ce cas je vous 	invite a faire un clique droit sur le fichier puis allez vers →propriété → permission → et 	cocher la case autoriser l’exécution du fichier comme un programme .

2. faire rentrer les information de jeux notamment la surface de placement des îles et le nombre d’îles, a noter que le nombre d’îles devra obligatoirement être inférieure ou égale a superficie.

3. placer les îles en sectionnant l’île avec un nombre de ponts que peut supporter cette île puis cliquer sur l’espace blanc pour la positionner, en cas d’erreur un autre clique sur l’île l’effacera de cette position, (il se peut  que les îles apparenterons après le positionnement de deux ou trois îles cela est du a un petit problème de performance mais rien de méchant (-₋-).

4. une fois toutes les îles placé le bouton « solve !» sera activer vous devriez cliquer sur ce dernier pour que l’algorithme de l’application commence a chercher une solution, si il trouve une il affichera une fenêtre qui vous informera qu’il a trouver une solution et il relira alors les îles par les ponts adéquats si non une fenêtre disant qu’aucune solution n’est possible et aucun ponts ne sera alors dessiné
